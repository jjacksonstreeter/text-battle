
# Text Battle (Streamlit)

Play a turn-based text battle in your browser.

## Run locally
```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate

pip install -r requirements.txt
streamlit run app.py
```

## Deploy (Streamlit Community Cloud)
1. Push this repo to GitHub
2. Create a new app in Streamlit Cloud pointing to `app.py`
3. Share the live URL in your portfolio

## Portfolio (GitHub Pages)
1. Create `docs/index.md` (included) and enable GitHub Pages from `/docs`.
2. Add your Streamlit live URL to the page.
